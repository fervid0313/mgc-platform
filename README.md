# MGS - Mind · Grind · Scale

A collaborative trading journal and community platform for day traders, swing traders, investors, and e-commerce entrepreneurs.

## Features

- **Real-time Trading Journal** - Log trades with profit/loss tracking
- **Community Spaces** - Join specialized groups (Day Trading, e-Commerce, etc.)
- **Friend System** - Connect with other traders
- **Mobile-First Design** - Optimized for all devices
- **Cloud Sync** - All data stored securely in Supabase

## Tech Stack

- **Frontend:** Next.js 16, React 19, TypeScript
- **Styling:** Tailwind CSS, Radix UI
- **Backend:** Supabase (PostgreSQL, Auth, Real-time)
- **Deployment:** Vercel

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (see `.env.local.example`)
4. Run database migrations in Supabase
5. Start development: `npm run dev`
